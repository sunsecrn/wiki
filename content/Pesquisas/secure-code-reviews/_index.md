## Secure Code Reviews

Grupo de estudos e prática focado em revisão segura de código, com o objetivo de formar especialistas em desenvolvimento seguro e análise de vulnerabilidades em código.

Objetivos do grupo:

- Encontrar vulnerabilidades de alto impacto em projetos open source
- Buscar CVEs de alto nível
- Gerar conteúdo técnico para a comunidade
- Desenvolver ferramentas relacionadas ao tema

Aqui concentramos conteúdos de:

- Guias práticos de ferramentas
- Checklists e fluxo de análise
- Material de estudo para quem está começando

Pré-requisitos: base de conceito de linguagem de programação, conhecimento em mais de uma linguagem para troca rápida de contexto, Git (principalmente), noções básicas de CodeQL e secure code review básico.

Seleção: fechado.

Encontro: toda quinta-feira à noite.

Dedicação: 5 horas por semana.

Dinâmica: estudos assíncronos durante a semana e alinhamento na reunião.

Ferramentas e plataformas:

- CodeQL
- SARIF Viewer
- Git
- SunCVE
